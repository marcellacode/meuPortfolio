# Portfólio - Marcella Soares

Portfólio profissional com foco em UX/UI, front-end e identidade visual pessoal.

## Stack atual (profissionalizada)

- Vite (build moderno e rápido)
- TypeScript (tipagem estática)
- ESLint (qualidade de código)
- HTML5 + CSS3 + Font Awesome

## Como rodar localmente

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev`: ambiente local
- `npm run build`: build de produção (com checagem TypeScript)
- `npm run preview`: preview do build
- `npm run lint`: validação de código
